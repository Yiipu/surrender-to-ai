Annex: Verification

To verify the timestamp proof, you may use the reference client commands below.

pip3 install opentimestamps-client
ots upgrade <file>.ots
ots verify <file>.ots <file>.json

Note: when a proof is freshly created, it may need time and/or upgrading before full verification succeeds.
